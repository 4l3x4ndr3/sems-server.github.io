Detailed Description

Interface for SIP signaling plugins that change requests or replies using hooks (ex: session timer).

From this type of plugins, an AmSessionEventHandler can be added to an AmSession. AmSessionEventHandler functions are called as hooks by the Session's event handler function.

Signaling plugins must inherit from this class.
