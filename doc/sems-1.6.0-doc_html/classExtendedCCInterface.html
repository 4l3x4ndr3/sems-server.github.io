virtual int ExtendedCCInterface::relayEvent ( SBCCallLeg *  call,
AmEvent *  e 
) [inline, virtual]

Possibility to influence messages relayed to the B2B peer leg. return value:

  • lower than 0 means error (returned upstream, the one returning error is responsible for destrying the event instance)
  • greater than 0 means "stop processing and return 0 upstream"
  • equal to 0 means "continue processing"

Reimplemented in CCDSMModule.
